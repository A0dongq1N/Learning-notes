# muduo

## muduo 安装

> 以centos7,gcc4.8.5 环境为例

### cmake安装

```shell
yum install cmake
```

### boost安装

```shell
yum install boost
yum install boost-devel
yum install boost-doc
```

### g++

```
yum install gcc-c++
```

### curl

```
yum install curl
```

### c-ares DBS

```
yum install c-ares-devel openssl-devel
```

### Google Protobuf

```

```
